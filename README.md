# Forest Fire Prevention

## Overview
Forest Fire Prevention is a web application designed to predict the probability of forest fire occurrences based on environmental factors such as temperature, oxygen levels, and humidity. The application is built using **Materialize CSS** for a modern UI and supports a dark mode for better visibility.

## Features
- **Forest Fire Prediction** based on user-input parameters.
- **Dark Mode UI** with Materialize CSS.
- **Animated Probability Output** for a dynamic user experience.
- **Responsive Design** for better accessibility.

## Technologies Used
- **HTML, CSS, JavaScript** (Frontend)
- **Materialize CSS** (UI Framework)
- **Flask (Backend, if applicable)**
- **Git/GitHub** (Version Control)

## Usage
1. Enter **temperature**, **oxygen levels**, and **humidity**.
2. Click **"Predict Probability"**.
3. The predicted probability of a forest fire is displayed dynamically.
